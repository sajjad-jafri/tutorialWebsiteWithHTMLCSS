Lists

List is a data structure that is used to store a collection of items. Lists are ordered, mutable, and can contain elements of different types. Duplication of elements is alowed in list.

To create a list, you can use square brackets [] and separate the elements with commas.


Here is an example of creating a list in Python.

Python Code

my_list = [1, 2, 3, 'apple', True]

In the above example, 'my_list' is a list that contains integers, string, and a boolean value.




Accessing elements in a List

Elements can be accessed in a list by their index. Remember that Python uses zero-based indexing, which means the first element in the list has an index of 0.

my_list = [1, 2, 3, 'apple', True]

0 1 2 3 4
1 2 3 apple True
-5 -4 -3 -2 -1


Here is an example of accessing elements in a list.

Python Code

my_list = [1, 2, 3, 'apple', True]

# Accessing Single Elements
print("First Element: ",my_list[0])

print("Third Index Element: ",my_list[3])

print("Last Element: ",my_list[-1])

# Accessing slice from a list
print("Slice of elements: ",my_list[1:3])

Output

First Element: 1

Third Index Element: apple

Last Element: True

Slice of elements: [2, 3]


You can also modify elements in a list by assigning new values to specific indexes:

Python Code

my_list = [1, 2, 3, 'apple', True]

# Modifying Element
my_list[1] = 5

print(my_list)

Output

[1, 5, 3, 'apple', True]


List Operations

Some common list operations include:

  • Concatenation

  • Replication

  • Append

  • Extend

  • Insert

  • Remove

  • Pop

  • Sort

  • Reverse



Concatenation

  • Process of combining two or more lists into a single list. In Python, you can concatenates lists using the + operator.


  • Python Code Example

  • list1 = [1, 2, 3]
list2 = [4, 5, 6]

concatenated_list = list1 + list2
print(concatenated_list)

  • Output

  • [1, 2, 3, 4, 5, 6]


Replication

  • Process of creating multiple copies of a list or its elements. In Python, you can replicate a list using the * operator.


  • Python Code Example

  • # replicating a single element
my_element = 5
replicated_element = [my_element] * 3
print("Replicated a single element: ",replicated_element)

# replicating list's elements
my_list = [1, 2, 3]
replicated_list = my_list * 2
print("Replicated elements of a list: ",replicated_list)

# replicating a list with different numbers of copies
my_list = [1, 2, 3]
replicated_list = [my_list] * 2
print("Replicated list: ",replicated_list)

  • Output

  • Replicated a single element:  [5, 5, 5]

Replicated elements of a list:  [1, 2, 3, 1, 2, 3]

Replicated list:  [[1, 2, 3], [1, 2, 3]]

    Note: replication creates a new list and does not modify the original list.




Append

  • It adds a single element to the end of the list. It does not return the new list, just modifies the original

  • Python Code Example

  • my_list = [10, 20, 30]

my_list.append(50)
print(my_list)

  • Output

  • [10, 20, 30, 50]



Extend

  • It adds multiple elements to the end of list.


  • Python Code Example

  • my_list = [10, 20, 30]

my_list.extend([50, 60, 70])
print(my_list)

  • Output

  • [10, 20, 30, 50, 60, 70]



Insert

  • Adding an element at a specific position.


  • Python Code Example

  • my_list = [1, 2, 3, 4]

my_list.insert(2, 200) # Will add 200 at index 2

print(my_list)

  • Output

  • [1, 2, 200, 3, 4]



Remove

  • Remove the first occurrence of an element. It throws error if element is not present.


  • Python Code Example

  • my_list = [1, 2, 3, 4, 2]

my_list.remove(2)

print(my_list)

  • Output

  • [1, 3, 4, 2]



Pop

  • Removes and returns the element at given index. Index is optional; if skipped, last element will be deleted.


  • Python Code Example

  • my_list = [10, 20, 30, 40, 50, 60]
my_list.pop(2)
print(my_list)

my_list.pop() # No index is given. It will remove last element
print(my_list)

  • Output

  • [10, 20, 40, 50, 60]

[10, 20, 40, 50]



Sort

  • sort the list in-place.


  • Python Code Example

  • my_list = [56, 37, 30, 84, 9, 27]
my_list.sort()
print(my_list)

  • Output

  • [9, 27, 30, 37, 56, 84]



Reverse

  • reverse the list in-place


  • Python Code Example

  • my_list = [56, 37, 30, 84, 9, 27]
my_list.reverse()
print(my_list)

  • Output

  • [27, 9, 84, 30, 37, 56]
