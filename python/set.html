Sets

A set is a data structure in Python that is:

  • Unordered: Sets do not maintain the order of elements.

  • Mutable: Sets can be modified after creation. While the set itself is mutable, the elements contained within the set must be of an immutable type, meaning my_set = {1,2, [3,4]} witll throw an error as list is mutable.

  • Unindexed: Since sets are unordered, so you cannot access or sliced elements by index like lists or tuples.

  • Iterable: Sets can be iterated over


Sets are defined by enclosing elements in curly braces {} or using the set() function.


Example

my_set = {1, 2, 3, 4, 5}


Sets are useful when:

  • You need to store a collection of unique elements (sets automatically remove duplicates).

  • You need to remove duplicates from a list.

  • You need to perform set operations like union, intersection, or difference.



How to declare empty set


Here is an example of declaring empty set

Python Code

my_set = {}
print(type(my_set))

Output

class 'dict'

What we can see here, with empty curly braces {}, we got the type - dictionary


For empty set we can use set() function.

Python Code

my_set = set()
print(type(my_set))

Output

class 'set'



Set Operations

  • Union: Combine elements from two sets (set1 | set2). "+" is unsupported in set.

  • Intersection: Find common elements between two sets ( set1 & set2).

  • Difference: Find elements in one set but not in another (set1 - set2).

  • Symmetric difference: Find elements in either set but not both (set1 ^ set2).


  • Python Code

    set1 = {1, 2, 3}
set2 = {4, 5, 3}

print("set1: ",set1)
print("set2: ",set2)

union_res = set1 | set2   
print("Union of set1 and set 2: ", union_res)

intersection_res = set1 & set2
print("Intersection between set1 and set2: ", intersection_res)

diff_res = set1 - set2
print("Difference in set1 to set2: ",diff_res)

Symdiff_res = set1 ^ set2
print("Elements which are not common from both set: ",Symdiff_res)

    Output

    set1:  {1, 2, 3}
set2:  {3, 4, 5}

Union of set1 and set 2:  {1, 2, 3, 4, 5}

Intersection between set1 and set2:  {3}

Difference in set1 to set2:  {1, 2}

Elements which are not common from both set:  {1, 2, 4, 5}



Adding and Removing element from a set

Since set is Unindexed, so we cannot access the elements but we can still add and remove the elements


Below is the python code which demonstrate how to add and remove elements from a set

Python Code

my_set = {10, 20, 30, 40, 50, 60, 70, 80}

print(my_set[0])    # type error
my_set[1] = 200     # type error

# Adding element
my_set.add(100)
print(my_set)

# Removing element
my_set.remove(70)
print(my_set)

Output

TypeError: 'set' object is not subscriptable
TypeError: 'set' object does not support item assignment

{100, 70, 40, 10, 80, 50, 20, 60, 30}

{100, 40, 10, 80, 50, 20, 60, 30}

Note: append() doesn't work in set. Since append adds the element at end but set is Unindexed.
