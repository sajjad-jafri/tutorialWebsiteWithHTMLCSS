Dictionary

A dictionary is a data structure in python that stores a collection of key-value pairs. It is an Unordered collection of items, where each item is a pair of a unique key and a value.

  • Keys: Must be Unique and Immutable for each item. It can be strings, numbers, or tuples. Keys cannot be List(mutable), Dictionary(mutable), Sets(Mutable).

  • Values: Associated data for each key. Values can be any python object.


Example

person = {'name':'John', 'age':30, 'city':'New York'}

Dictionaries are:

  • Mutable: Can add, remove, or modify key-value pair.

  • Dynamic: Can grow or shrink as needed.

  • Fast lookups: Efficiently retrieve values by their keys.


Dictionaries are useful for:

  • Data storage and retrieval

  • Configuration files

  • JSON data handling

  • Counting and grouping data




Common dictionaries operations

  • Create: my_dict = {'key': 'value'}

  • Access: my_dict['key']

  • Update: my_dict['key'] = 'new_value'

  • Delete: del my_dict['key']

  • Check existance: 'key' in my_dict




Creating Dictionaries


Empty dictionary

  • my_dict = {}

  • my_dict = dict()


  • We can insert key-value pair later after creating empty dictionary using my_dict[key] = value

    • my_dict = {}

my_dict['name'] = 'John'
my_dict['age'] = 30
	
print(my_dict)

    Output

    {'name': 'John', 'age': 30}

Initialize with key-value pairs

  • my_dict = {'name': 'John', 'age': 30}

  • my_dict = dict(name='John', age=30)


From a list of tuples

  • my_dict = dict([('name', 'John'), ('age', 30)])


Using dictionary comprehension

  • my_dict = {x: x**2 for x in range(5)}

    • {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}




Accessing elements from dictionary


Using Key

  • Elements can be accessed through key, you will get keyerror if key is not present.

  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}

print(my_dict['b'])

    Output

    banana



Using get() method

  • Using get() method, we don't get keyerror if key is not present instead you will get the default value.

  • Syntax: my_dict.get(key, default_value). default_value is optional.


  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}

print(my_dict.get('b'))

print(my_dict.get('d', "key is not present, Sorry!"))

print(my_dict.get('e'))	 # When default-value and key both not present

    Output

    banana

key is not present, Sorry!

None



Accessing multiple values


using keys() method

  • returns a view object with all keys

  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}

print(my_dict.keys())

for k in my_dict.keys():
    print(k)

    Output

    dict_keys(['a', 'b', 'c'])

a
b
c


using values() method

  • returns a view object with all values

  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}

print(my_dict.values())

for v in my_dict.values():
    print(v)

    Output

    dict_values(['apple', 'banana', 'cat'])

apple
banana
cat


using items() method

  • returns a view object with key-value pairs

  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}

print(my_dict.items())

for k,v in my_dict.items():
    print(k,"....",v)

    Output

    dict_items([('a', 'apple'), ('b', 'banana'), ('c', 'cat')])

a .... apple
b .... banana
c .... cat



Updating dictionaries


Update a single key-value pair

  • my_dict[key] = value

  • If key is not present - it adds a new record

  • If key is already present - it will update the old value


  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}

my_dict['b'] = 'bat'
my_dict['d'] = 'dog'

print(my_dict)

    Output

    {'a': 'apple', 'b': 'bat', 'c': 'cat', 'd': 'dog'}


Update multiple key-value pair

  • can update multiple key-value pair using update() method


  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}

my_dict.update({'b': 'bat', 'c': 'cricket'})

print(my_dict)

    Output

    {'a': 'apple', 'b': 'bat', 'c': 'cricket'}



  • update using multiple dictionary

  • Python Code

    • dict1 = {'a': 'apple', 'b': 'banana', 'c': 'cat'}

dict2 = {'d': 'dog', 'e': 'egg'}
dict1.update(dict2)

print(dict1)

    Output

    {'a': 'apple', 'b': 'banana', 'c': 'cat', 'd': 'dog', 'e': 'egg'}



Deleting/Removing element from the dictionary


del my_dict[key]

  • If key is present - will delete the key-value pair.

  • If key is not present - keyError


    • Python Code

    my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}
print("Original dictionary: ",my_dict)

del my_dict['b']

print("After deleting key: ",my_dict)

del my_dict['d']	# 'd' is not present

    Output

    Original dictionary:  {'a': 'apple', 'b': 'banana', 'c': 'cat'}

After deleting key:  {'a': 'apple', 'c': 'cat'}

KeyError: 'd'



my_dict.clear()

  • It will delete all content of the dictionary not the dictionary meaning empty dictionary will remail available with same name


  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}
print("Original dictionary: ",my_dict)

my_dict.clear()

print("After clearing the content: ",my_dict)

    Output

    riginal dictionary:  {'a': 'apple', 'b': 'banana', 'c': 'cat'}
After clearing the content:  {}



del my_dict

  • It will delete the dictionary.


  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}
print("Original dictionary: ",my_dict)

del my_dict

print(my_dict)

    Output

    Original dictionary:  {'a': 'apple', 'b': 'banana', 'c': 'cat'}

NameError: name 'my_dict' is not defined



pop() method

  • Python Code

    • my_dict = {'a': 'apple', 'b': 'banana', 'c': 'cat'}
print("Original dictionary: ",my_dict)

my_dict.pop('b', "key not present")

print(my_dict)

    Output

    Original dictionary:  {'a': 'apple', 'b': 'banana', 'c': 'cat'}

{'a': 'apple', 'c': 'cat'}
