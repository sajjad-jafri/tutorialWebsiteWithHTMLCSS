Tuples

A tuple is a data structure in Python that similar to a list, but it is immutable, meaning its contents cannot be modified after creation.

Tuples are defined by enclosing elements in paranthesis () instead of square brackets [] like lists.


Here is an Example of tuple

PythonCode

my_tuple = (1, 2, 3)


Tuples are useful when:

  • You need to store a collection of data that shouldn't be changed.

  • You need to use a collection as a dictionary key (tuples can be used as keys, but lists can't).

  • You need to return multiple values from a function (tuples can be used to return multiple values).



Accessing elements in a tuple


Python Code

my_tuple = (1, 20, 30, 4, 'boy', 100)
print(my_tuple[0])

print(my_tuple[-1])

print(my_tuple[1:4])

Output

1

100

(20, 30, 4)


We cannot modify tuple element as it is immutable

Python Code

my_tuple = (1, 2, 3)
my_tuple[1] = 20

Output

TypeError: 'tuple' object does not support item assignment

Similary append() and remove() also doesn't work in tuple




Basic tuple Operations


Python Code

my_tuple = (1, 2, 3)

print(len(my_tuple))

tuple2 = (4,5)

# Concatenation
print(my_tuple + tuple2)

#Replication
print(my_tuple * 3)

# in operator
print(3 in my_tuple)

Output

3

(1, 2, 3, 4, 5)

(1, 2, 3, 1, 2, 3, 1, 2, 3)

True
