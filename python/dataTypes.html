Data Types in Python

A data type is a classification of data which tells the compiler/interpreter that how the programmer intends to use the data.

Some built-in Python data types are:




  • int, float, complex, bool, str are fundamental data types.

  • All fundamental data typess are immutable.

  • we will study about immutability more later in this section.

  • In Python, we don't need to specify the type of the value, Python automatically finds the type of value you assign.

Complex data type

In Python, the complex data type represents complex numbers, which consist of a real part and an imaginary part.

  1. Representation:

    • A complex number is denoted as A + Bj or A + Bi, where A is the real part and B is the imaginary part.

    • Python uses the A + Bj notation for complex numbers (e.g., 3 + 4j), where j represents the square root of -1.
      Note that 3 + 4i is not valid in Python.




  2. Creating Complex Numbers:

    • You can create a complex number from two real numbers using the complex(a, b) function, where a and b are real values.



  3. Accessing Real and Imaginary Parts:

    • After creating a complex number, you can access its real and imaginary parts using the built-in data descriptors:

      • a.real returns the real part (e.g., 5.0 for 5 + 6j).

      • a.imag returns the imaginary part (e.g., 6.0 for 5 + 6j).



  4. Conjugate of Complex Numbers:

    • The conjugate() method returns the conjugate of a complex number.

    • Example: If a = 10 - 6j, then a.conjugate() results in (10 + 6j).



  5. Operations on Complex Numbers:

    • Python supports direct arithmetic operations on complex numbers using operators (+, -, *, /).

    • Example:

      • Addition: (1 + 2j) + (3 + 4j) results in (4 + 6j)

      • Subtraction: (1 + 2j) - (3 + 4j) results in (-2 - 2j)

      • Multiplication: (1 + 2j) * (3 + 4j) results in (-5 + 10j)

      • Division: (1 + 2j) / (3 + 4j) results in (0.44 + 0.08j)





Boolean data type

In Python, the boolean data type represents one of two values: True or False.

  1. Boolean Values:

    • A boolean variable can hold either True or False.



  2. Evaluating Variables and Expressions:

    • This function converts a value to a boolean (True or False) based on the standard truth testing procedure.



  3. Integers and Floats as Booleans:

    • Any integer, floating-point number, or complex number with a value of zero is considered False.

    • Non-zero values are considered True.





String data type

In Python, the string data type represents text data. It’s a sequence of characters enclosed in single quotes, double quotes, or triple quotes. Strings are immutable, meaning once created, they cannot be modified directly.

E.g.,
s = 'Hello'





Bytes data type

In Python, the bytes data type represents a sequence of integers in the range of 0 to 255.

A bytes object is immutable, meaning once it’s created, its contents cannot be changed.







Bytearray data type

Same as bytes data type, the only difference is bytearray is mutable.







Range data type

The range() function returns an immutable sequence of numbers. By default, it starts from 0 and increments by 1, stopping before a specified number.

Some key points:

  1. Parameters:

    • start (optional): An integer specifying the starting position (default is 0).

    • stop (required): An integer specifying the stopping position (not included).

    • step (optional): An integer specifying the incrementation (default is 1).



  2. Examples:

    • Create a sequence from 0 to 3: x = range(4)



    • Create a sequence from 3 to 5: x = range(3, 6)



    • Create a sequence from 3 to 9 (increment by 2): x = range(3, 10, 2)






List data type

A list is an ordered collection of elements of any data type between square brackets [].

Mutability: Lists are mutable, meaning you can modify them by adding, removing, or changing elements.

Duplication: Duplication is allowed, meaning you can have same multiple items.

Heterogeneous: Lists can contain various data types, such as integers, floats, strings, or even other lists.

E.g., l = [10, 'John', 2.5]



Tuple data type

A tuple is an ordered collection of items enclosed in round brackets. Unlike lists, tuples cannot be changed once created, meaning tuples are immutable.

Tuples are same as lists but tuples are immutable.

E.g., t = (10, 'd', 34.2)





Set data type

Set is an unordered collection of unique items represented inside curly braces {}.

Set is mutable.

Indexing or slice operator is not allowed since there is no guarantee in Set that which element is at which position because oder in set is not preserved.

E.g., s = {10, 20, 30)
