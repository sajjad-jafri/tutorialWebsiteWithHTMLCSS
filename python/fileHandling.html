File Handling

Python supports file handling which allows users to read, write, and manipulating files. Python provides built-in functions and modules to interact with files, allowing you to:

  • Opening a file

  • Read data from files

  • Write data to files

  • Append data to files

  • Delete files

  • Rename files

  • Check the existance

  • Get files information (size, modification time, etc.)




Opening a file

Syntax:

open(filename, mode)

It will return a file object


Modes can be one of the following:

  • r:

    • Open an existing file for read operation.

    • f = open('abc.txt') OR f = open('abc.txt', 'r')

    • This is the default mode, means if no mode is specified, it will be selected automatically.

    • You will get FileNotFoundError if file is not present.


  • w:

    • Open a file for write operation.

    • f = open('abc.txt', 'w')

    • If the file doesn’t exist, it creates a new one.

    • If the file exists, It will overwrite (clears) an existing data.


  • a:

    • Open a file for append operation.

    • f = open(abc.txt', 'a')

    • If the file doesn’t exist, it creates a new one.

    • The file pointer is at the end, so you can add content to the existing file - means old data will not be cleared.


  • r+:

    • read and write

    • f = open('abc.txt', 'r+')


  • w+:

    • write then read


  • a+:

    • append and read


  • x:

    • Open a file in exclusive mode for write operation.

    • f = open('abc.txt', 'x')

    • Specified file should not be present, if file is already present - FileExistsError.

    • It will create a new file.

    • Same as 'w' mode except file should not be available.




Reading files

To read a file we can use any of the below method:

  • read() - reads the entire file content.

  • readline() - reads a single line.

  • readlines() - reads all lines into a list.


Use close() method to close the file object after reading.

Alternatively, use a with statement to automatically close the file.


Lets understand it with an example


abc.txt

Lorem Ipsum is simply dummy text 
of the printing and typesetting 
industry. Lorem Ipsum has been 
the industry's standard dummy 
text ever since the 1500s, when 
an unknown printer took a galley 
of type and scrambled it to make
a type specimen book.

Python Code

file = open('abc.txt', 'r')
content = file.read()
file.close()
print(content)

Output

of the printing and typesetting 
industry. Lorem Ipsum has been 
the industry's standard dummy 
text ever since the 1500s, when

a type specimen book.


Using with statement

Python Code

with open('abc.txt', 'r') as file:
    content = file.read()
print(content)

Reading single line

Python Code

with open('abc.txt', 'r') as file:
    line1 = file.readline()
    line2 = file.readline()	# To read next line
print(line1)
print(line1)

Reading all lines into a list

Python Code

with open('abc.txt', 'r') as file:
    lines = file.readlines()
print(lines)

Writing data to files

Writing to a file can be achieved by two methods:

  • write(str) - writes a single string.

  • writelines(list of lines) - writes a list of strings.


Python Code

with open('abc.txt', 'w') as file:
    file.write('Hello')
    file.write('Learn')
    file.write('Python\n')
    file.write('C++\n')
    file.write('Java')

If the file exists, it truncates (clears) its content before writing, as the pointer is at beginning for 'w' mode. If the file doesn’t exist, it creates a new one.


abc.txt

HelloLearnPython
C++
Java


writing multiple lines

Python Code

with open('abc.txt', 'w') as file:
    lst = ["Python\n", "Scala\n", "R"]
    file.writelines(lst)

abc.txt

Python
Scala
R


Exclusive mode

'x' mode opens a file for exclusive creation.

Sometimes 'w' mode can raise a problem as it clears out the existing data, so for this solution we have 'x'

In exclusive mode - if the file already exists, it raises a polite little FileExistsError exception.

Essentially, it dont want mess with already existing and creates a new file if it is already not created.


Python Code

with open('abx.txt', 'x') as file:
    file.write("Text example")



Append data to files

We can append text to a file in Python with 'a' mode.

'w' mode clears out the existing data because file pointer is at the beginning.

'a' is used to append the data, the file pointer is the end so it doesnot clears the existing data.



Python Code

with open('abc.txt', 'a') as file:
    file.write("Text to append")

OS

The OS module in Python provides a way to interact with the operating system and perform various file handling tasks. Here are some common os module functions for file handling:

  • os.getcwd(): Returns the current working directory

  • os.chdir(path): Changes the current working directory to path.

  • os.listdir(path): Returns a list of files and directories in path.

  • os.mkdir(path): Creates a new directory at path.

  • os.rmdir(path): Removes an empty directory at path. you will get error if any file or directory inside the path exists. To delete a directory which is not empty - shutil module will be used.

  • os.rename(src, dst): Renames a fileor directory from src to dst.

  • os.remove(path): Deletes a file at path.

  • os.path.join(path1, path2, ...): Joins multiple path components into a single path.

  • os.path.exists(path): Checks if a file or directory exists at path.

  • os.path.isfile(path): Checks if path is a file.

  • os.path.isdir(path): Checks if path is a directory.

  • os.path.getsize(path): Returns the size of a file at path.



Python Code

import os

# if new_dir exists, Removing new_file (inside new_dir)
if os.path.exists("new_dir"):
    os.remove("new_dir/new_file.txt")
else:
    # Create a new directory
    os.mkdir("new_dir")

    # Change into the new directory
    os.chdir("new_dir")

    # List files and directories
    print("Files and Directories under new_dir: ",os.listdir())

    # Create a new file
    with open("new_file.txt", "w") as f:
        f.write("Hello")

    # Check if the file exists
    print("Is file exists: ",os.path.exists("new_file.txt"))

    # Get the file size
    print("File Size: ",os.path.getsize("new_file.txt"))

    # Remove the file
    os.remove("new_file.txt")

    # Remove the directory
    os.chdir("..")  # back to main directory
    os.rmdir("new_dir")

Output

Files and Directories under new_dir:  []
Is file exists:  True
File Size:  5


shutil

The shutil module in python provides high-level file handling operations, including:

  • Copying files and directories

  • Moving files and directories

  • Removing files and directories

  • Archiving files


Common shutil functions:

  • shutil.copy(src, dst): Copies a file from src to dst.
