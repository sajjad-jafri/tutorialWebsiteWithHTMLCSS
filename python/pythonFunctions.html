Functions

Function is a block of code that perform a specific task and can be reused in a program.

To define a function in Python, def keyword is used, followed by the function name and parentheses.

This block of code runs when it is called.

A function can return data as a result.


Syntax

def function_name([parameters]):     # Parameters are optional
    ----staments----
    ----------------

Python Code Example

def greet(name):	# Function with argument
    print("Hello, " + name + "!")

# Call the function with different arguments
greet("Alice")
greet("Bob")
greet("Charlie")

Output

Hello, Alice!
Hello, Bob!
Hello, Charlie!


How Python executes a function:

Statement - 1

  • def greet(name): - It is a function header


Statement - 2

  • print("Hello, " + name + "!") - It is body of a function or a definition.

  • The statements inside a function-body are not executed; this gets executed only when the function is called


Statement - 3 to 5

  • greet("Alice") - Its a function call. When a function calls encounters, Python jumps to function definition and start executing it.

  • It will take "Alice" as parameter to function header and "Alice will be assigned to name.

  • Similary statements 4 and 5 will execute.




Nested Functions

A nested function is a function defined inside another function.


Python Code Example

def greeting(first, last):
  # nested helper function
  def getFullName():
    return first + " " + last

  print("Hi, " + getFullName() + "!")

greeting('Quincy', 'Larson')

Output

Hi, Quincy Larson!

Arguments and Parameters


Arguments: Values which are being passed. Appear in function call.

Parameters: Values which are recieved. Appear in function header.

def multiply(x, y):	# x and y are parameters
    return x*y

a, b = 3, 4 
m = multiply(a, b)	# a and b are arguments
print(m)

From above example: x and y are Parameters. a and b are arguments


Python support 3 types of parameters

  • Positional parameters

  • Default parametrs

  • Keywords(or named) parameters



Positional parameters

def check(a, b, c):
    ......
    ......

check(2, x, y)

2 → a,
x → b,
y → c

argument must be provided for all parameters.




Default parameters

Python allows us to assign default values to a function parameter.

This is useful when we already know the calue of certain parameters.


Python Code

def greet(name, message = "Good Morning!"):	# message is a default parameter
    print(f"Hello, {name}! {message}")

greet("John")

Output

Hello, John! Good Morning!

Here,

message is Default Parameter

"Good Morning" is Default argument

name is required parameter


From the above code, we did not pass the value for message. This means that if the argument is not provided when the function is called, it will use the default value.


What if we pass the value for default parameter?

Python Code

def greet(name, message = "Good Morning!"):	
    print(f"Hello, {name}! {message}")

greet("John", "How are you")

Output

Hello, John! How are you

In a function header, any parameter cannot have a default value unless all parameters appearing on its right have their default values.

  • def interset(prin, time, r=0.10): → Valid

  • def interset(prin, time=2, r): → Invalid

  • def interset(prin=5400, time=2, r): → Invalid

  • def interset(prin, time=2, r=0.10): → Valid

  • def interset(prin=2000, time=2, r=0.10): → Valid

Lambda Function

A lambda function, also known as an anonymous function(defined without a name), is a compact, self-contained block of code that can take any number of arguments and evaluate a single expression.


Syntax

lambda arguments: expression


lambda x, y: x + y is similar to

def(x, y)
  return x + y


Python Code

add = lambda x, y: x + y
print(add(7,3))

Output

10


Lets take a look at one more program lambda with map

Python Code

numbers = [1,2,3,4]
squared_num = list(map(lambda x: x**2, numbers))
print(squared_num)

Output

[1,4,9,16]
