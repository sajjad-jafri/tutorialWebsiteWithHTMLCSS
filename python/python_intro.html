What is Python?

Python is a high-level, interpreted, object oriented programming language for genral purpose programming. It is known for its simplicity and readability. It was created by Guido Van Rossum and first released in 1991.

It is widely used in various fields such as web development, data analysis, artificial intelligence, and more.

Tokens in Python

Tokens or lexical units are the smallest fractions in the python programme. There are 5 types of tokens in python which are listed below:

  1. Keywords

    1. These are reserved words that has a specific meaning. There are 33 keywords used in version 3.7.

    2. Ex: del, True, if, while, break, etc.

  2. Identifiers

    1. Various names of variables, functions, list, dictionary, etc.

  3. Literals

    1. Constant values. Python supports a variety of literals:

      • String literals

      • Numeric literals

      • Boolean literals

      • Literal Collection (Lists, Tuples, Dictionaries, and Sets)

  4. Operators

    1. There are 7 different operators.

      1. Arithmetic Operators (+, -, x, /, %)

      2. Relational Operators (==, !=, >, <)

      3. Assignment Operators (+=, -=)

      4. Logical Operators (AND, OR)

      5. Bitwise Operators (XOR or ^)

      6. Membership Operators

      7. Identity Operators

  5. Punctuators

    • Some commonly used punctuators are: ‘, #, \ ,( ) ,{ },[ ] ,@ ,: , =


Comments

Python comments are simple sentences that we use to make the code easier to understand. It is ignored by the interpretor.

  • Single-Line Comments

    • Single-line comments begin with the “#” character.

  • Multi-Line Comments

    • Python does not support multi-line comments. However, there are multiple ways to overcome this issue. None of these ways are technically multi-line comments, but you can use them as one. The first way is by using # at the beginning of each line of the comment.

    • The next way is by using string literals but not assigning them to any variables. If you do not assign a string literal to a variable, the Python interpreter ignores it. Use this to your advantage to write multi-line comments. You can either use a single (‘’) quotation or double (“”) quotation.


Variables

Python gives del statement to remove a variable.

In Python, variable do not need declaration. Python create a variable of the type similar to the type of value assigned

E.g., a = 70   // a is numeric variable
