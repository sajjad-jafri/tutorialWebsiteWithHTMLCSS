Ways to create NumPy Arrays

Lets explore how to create NumPy Arrays

  • Transforming standard list

  • Built-in functions

  • Generating random data


Create Numpy Arrays Using Lists or Tuples

The simplest way to create a NumPy array is by passing a Python list or tuple to the numpy.array() function.

Python Code

	
  import numpy as np

  my_list = [1, 2, 3, 4, 5]
  numpy_array = np.array(my_list)

  print("NumPy Array using Python List:\n",numpy_array)

Output


NumPy Array using Python List:
[1 2 3 4 5]


For a matrix, we will use list of list as shown below using Python code

Python Code


  import numpy as np

  my_matrix = [[1,2,3],[4,5,6],[7,8,9]]	

  arr = np.array(my_matrix)

 print(arr)

Output


  [[1 2 3]
   [4 5 6]
   [7 8 9]]



Creating NumPy Array Using Built-in Functions

NumPy provides several built-in functions to generate arrays with specific properties.

  • np.arange(): Creates an array with values that are evenly spaced within a given range.

  • np.zeros(): Creates an array filled with zeros.

  • np.ones(): Creates an array filled with ones.

  • np.full(): Creates an array filled with a specified value.

  • np.linspace(): Creates an array with values that are evenly spaced over a specified interval.

  • np.eye(): Creates an identity matrix.

  • np.diag(): Constructs a diagonal array.


Python Code


# importing numpy module
import numpy as np

range_array = np.arange(0, 10, 2)  # start, stop, step
zeros_array = np.zeros((2, 3))      # num of row, num of column
ones_array = np.ones((3, 3))
constant_array = np.full((2, 2), 7)
linspace_array = np.linspace(0, 10, 3)  # start, stop, how many num between start and stop
identity_matrix = np.eye(4)   # num of row = num of column
diagonal_array = np.diag([1, 2, 3])

print("Range Array:","\n",range_array)
print()
print("Zero Array:","\n",zeros_array)
print()
print("Ones Array:","\n",ones_array)
print()
print("Constant Array:","\n",constant_array)
print()
print("Linspace Array:","\n",linspace_array)
print()
print("Identity Matrix:","\n",identity_matrix)
print()
print("Diagonal Array:\n",diagonal_array)

Output

 [0. 0. 0.]]

Ones Array:

 [1. 1. 1.]

Creating Numpy Arrays Using Random Number Generation

NumPy comes with its own random library and provides functions to create arrays filled with random numbers.

  • np.random.rand()

  • np.random.randn()

  • np.random.randint()


np.random.rand(): Creates an array of specified shape and fills it with random values sampled from a uniform distribution over [0, 1). Here 0 is inclusive and 1 is exclusive, meaning it can return back 0, but it can't return back 1.

Python Code


import numpy as np

random_single_num = np.random.rand(1)  # Single num to fall between 0 and 1

print("Random Single Num:\n",random_single_num)

# Can give the shape as well
another_random_5_by_2 = np.random.rand(5,2) # 5 rows, 2 columns 
print("5 Rows 2 Columns:\n", another_random_5_by_2)

Output


Random Single Num:  
[0.85518742]

5 Rows 2 Columns:
 [[0.81326532 0.39351299]
 [0.61477625 0.60763054]
 [0.38763695 0.55048548]
 [0.43550949 0.32708931]
 [0.26035038 0.84531517]]

If you run it again and again, it will give you always different result.

Notice all these numbers fall between 0 and 1, and it's from a uniform distribution. So all the numbers between 0 and 1 have the same likelihood of being returned.



np.random.randn(): Creates an array of specified shape and fills it with random values sampled from a standard normal distribution.


Python Code


import numpy as np

random_5_num = np.random.randn(5)  # Give 5 num from standard normal distribution

print("Random 5 number from standard Normal Distribution:\n",random_5_num)

# Arrang in shape
another_random = np.random.randn(2,3)
print("Another random num with shape 2 and 3:\n", another_random)

Output


Random 5 number from standard Normal Distribution:
 [ 0.61273118 -0.94130053 -0.65722781  0.40961921  0.64256434]

Another random num with shape 2 and 3:
 [[-2.11003834  2.60083029  0.68792849]
 [ 0.0760018  -0.94904411 -1.2672804 ]]

The standard normal distribution is one of the forms of the normal distribution. It occurs when a normal random variable has a mean equal to zero and a standard deviation equal to one. So if mean is 0 so the fact is we can get negative numbers.



np.random.randint(): Creates an array of specified shape and fills it with random integers within a given range.


Python Code


import numpy as np

# show 5 random numbers between 0(inclusive) to 101(exclusive)
random_int = np.random.randint(0, 101, 5)  # 5 is size here, not step like in range

print("Random 5 Integers between 0 to 101\n",random_int)

# Show random numbers between 0 to 101 in shape of 4(row) and 5(column)
random_int_4_by_5 = np.random.randint(0, 101, (4,5))
print("Random integers with shape 4 and 5:\n", random_int_4_by_5)

Output


Random 5 Integers between 0 to 101
 [36  0 27 10 83]

Random integers with shape 4 and 5:
 [[52 56 40 60 72]
 [22 77 83 51 26]
 [14 54 14 81 82]
 [47 33 72 79 22]]



Something to be note above, If you run any above programs we have discussed, every time you the run the random function program you will get the different result. random() function is used to generate random numbers in Python, so they can be random at every run.



random.seed( ) in Python

Seed function is used to save the state of a random function, so that it can generate same random numbers on multiple executions of the code on the same machine or on different machines (for a specific seed value).


Python Code


import numpy as np

np.random.seed(42)   # Any arbitrary number can be passed

r = np.random.rand(4)

print(r)

Output


[0.37454012 0.95071431 0.73199394 0.59865848]

When you run this same program multiple times, you will get the same output.




Reshaping an Array:

We can use reshape method to reshape an array.


Python Code


import numpy as np

arr = np.arange(20)
print("Array:\n", arr)

reshaped_array = arr.reshape(5, 4)
print("Reshaping Array with 5 by 4:\n", reshaped_array)

reshaped_array2 = arr.reshape(4,5)
print("Reshaping Array with 4 by 5:\n", reshaped_array2)

Some methods to remember

  • max() - to give the maximum element from the given array.

  • min() - to give the minimum element from the given array.

  • argmax() - to get the index location of the maximum element of the array.

  • argmin() - to get the index location of the minimum element of the array.

  • shape() - to get the dimension/shape of the array.


Python Code 

import numpy as np

arr = np.random.randint(0, 101, 10) 

print("Given Array\n", arr)

print("Maximum number\n", arr.max())
print("Minimum number\n", arr.min())
print("Index location of the maximum number\n", arr.argmax())
print("Index location of the minimum number\n", arr.argmin())
print("Its Dimension\n", arr.shape)  # Its attribut not function 
      
r_arr = arr.reshape(5,2)
print("Reshaping Array with 5 by 2:\n", r_arr)
print("New Dimension\n",r_arr.shape)
