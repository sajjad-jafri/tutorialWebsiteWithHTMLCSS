What is Numpy?

NumPy stands for Numeric python which is a python package for creating multidimensional (N-D) and single-dimensional (1-D) array elements.

Numpy has the ability to quickly broadcast functions, and is also used in linear algebra, Fourier transform, statistical distributions, trigonometric functions and random number of capabilities.

While NumPy structures look similar to standard Python lists, they are much more efficient! It provides an array object much faster than traditional Python lists.

NumPy provides a convenient and efficient way to handle the vast amount of data. NumPy is also very convenient with Matrix multiplication and data reshaping. NumPy is fast which makes it reasonable to work with a large set of data.

Numpy arrays essentially come in two flovors: Vectors and Matrices. Vectors are strictly 1-d arrays and matrices are 2-d (but you should note a matrix can still have only one row or one column).

You can install NumPy simply using pip command in your command prompt window using 'pip install numpy'




Types of Array:

  • One Dimensional Array

  • Multi-Dimensional Arra



One Dimensional Array:

A one-dimensional array is a type of linear array.

One Dimensional Array
1 2 3 4

Python Code


  # importing numpy module
  import numpy as np

  # creating list
  l = [1, 2, 3, 4]

  # Creating numpy array
  arr = np.array(l)

  print("List in Python: ", l)
  print("Numpy Array in Python: ", arr)

Output:

        
  List in Python:  [1, 2, 3, 4]
  Numpy Array in Python:  [1 2 3 4]



Multi-Dimensional Array:

Data in multidimensional arrays are stored in tabular form.

Two Dimensional Array
1 2 3 4
5 6 7 8

Python Code

	
  # importing numpy module
  import numpy as np

  # creating list
  list_1 = [1, 2, 3, 4]
  list_2 = [5, 6, 7, 8]

  # Creating numpy array
  arr = np.array([list_1, list_2])


  print("Numpy multi dimensional array in Python:\n ", arr)

Output

	
  Numpy multi dimensional array in Python:
    [[1 2 3 4]
   [5 6 7 8]]

Note: use [ ] operators inside numpy.array() for multi-dimensional
